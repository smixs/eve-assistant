Iva — персональный AI-агент с долговременной памятью

Универсальный персональный AI-агент с долговременной памятью. Который просто работает.

Живёт в Telegram. Ставится одной командой. Работает с любого IP.

Открыть на GitHub Как установить

Установка одной командой

curl -fsSL https://raw.githubusercontent.com/smixs/iva/main/install.sh | bash

Что умеет

  • 🎙️ Голос и видео - расшифровывает голосовые и кружки, понимает речь на любом языке.
  • 🧠 Долгая память - граф в виде дерева из обычных файлов, без тяжёлых баз данных.
  • 🔎 Быстрый поиск - находит нужное за секунды.
  • ⏰ По расписанию - дайджест дня или недели, напоминания, регулярные задачи. Может проверить почту и прислать сводку сам, вовремя.
  • 🤖 Выбор модели - DeepSeek, Kimi, GLM и другие открытые модели.
  • 🔒 Приватность - весь код и память у вас, не в чужом облаке.
  • 🛰️ Работает с любого IP - в том числе из России и Беларуси, без VPN.

Почему Iva

Завести личного агента с настоящей памятью обычно дорого и сложно, а топовые ассистенты доступны не везде. Iva ставится одной командой на любой сервер, работает на лучших открытых моделях и держит всю память у вас. Справится даже человек без технического бэкграунда.
