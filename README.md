## Что это

Iva — твой личный ИИ-агент на собственном сервере. Одна команда — установщик сам всё поставит и
проведёт по настройке. Провайдера выбираешь сам (бери самый дешёвый и доступный), рекомендуемая
модель — **DeepSeek-V4-Pro**. Готово — пиши боту в Telegram.

Память у Iva долговременная, в виде **дерева**: день → неделя → месяц → год. Дерево — как ива,
отсюда и имя.

## Умеет

- 🎙️ Голос и видео
- 🔎 Веб-поиск
- 🌐 Браузер
- ✅ Задачи

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/smixs/eve-assistant/main/install.sh | bash
```

Ответь на вопросы установщика (провайдер, ключи, Telegram-бот) — и пиши боту.
